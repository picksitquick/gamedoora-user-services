# gamedoora-user-services

The Spring boot microservice application for gamedoora

## Requirements
- JDK 1.8 or higher
- MySql 5 or higher
- Maven 3.3 or higher 

## Command to build and launch
```
mvnw clean install
```
```
mvnw spring-boot:run 
```
